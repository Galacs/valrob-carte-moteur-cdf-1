# Carte moteur CDF 1

ESP32-S3-WROOM-1 Dual 5A DC motor driver

## IC List

- ESP32-S3-WROOM-1 (Main MCU Module)
- AP2112K (MCU 3v3 LDO)
- DRV8874 (Integrated FETs H-bridge)
